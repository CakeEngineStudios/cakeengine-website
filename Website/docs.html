Entity()

The Entity class is a barbone section to rendering all things like cubes and models. This is what fills the scene of the graphical world with whatever you desire.

Entity(parent = scene, model = None, position = Vec3(0, 0, 0), **kwargs)

.model = = Noneset model as string (eg: 'cube', 'model_name.extension_name')
.name = self.__class__.__name__sets the name of the entity through string
.enabled = Trueif False, will not run update loops and will not render.
.ignore_paused = Falseif True, will still run when application is paused.

Examples

#creating a object through script
test_entity = Entity(model = 'cube', scale = Vec3(3, 4.3 ,1))
#change color of entityv using rgba function
test_entity.color = rgba(255, 0, 100, 0.7)
#change texture to internal texture using texture function
test_entity.texture = "brick"
